# Motorola IMPRES Battery Reader (ESP32)

Зчитування / програмування чипів акумуляторів Motorola IMPRES
(**DS2433** — EEPROM 512 Б, **DS2438** — паливомір) через веб-інтерфейс
Wi-Fi та OLED/LCD-дисплей. Інтерфейс українською, заставка з емблемою
Національної Гвардії України.

---

## 1. Що потрібно

**Бібліотеки (Arduino Library Manager):**
- `U8g2` (олеофрей) — дисплей
- `OneWire` — шина 1-Wire
- `ArduinoJson`

**Плата:** будь-яка ESP32 (пакет `esp32` від Espressif, перевірено на 3.3.10).
Прошивка займає ~79% флеша — підходить схема розділів за замовчуванням.

## 2. Підключення

| Сигнал | GPIO ESP32 | Примітка |
|---|---|---|
| 1-Wire дані (DS2433/DS2438) | **13** | `DS_PIN` |
| Керування підтяжкою 1-Wire | **12** | `PULLUP_PIN` (сильний pull-up на час tPROG) |
| LED зелений | **14** | |
| LED червоний | **27** | |
| Кнопка «Вперед» | **25** | між GPIO і GND, активний LOW |
| Кнопка «Назад» | **26** | між GPIO і GND, активний LOW |

Піни змінюються в **`settings.h`**.

## 3. Вибір дисплея (головне)

У **`settings.h`**, КРОК 1 — розкоментуйте **рівно один** рядок:

```c
#define DISPLAY_SSD1306_I2C        // 0.96" 128x64 I2C (GME12864) — за замовч.
// #define DISPLAY_SH1106_I2C      // 1.3"  128x64 I2C
// #define DISPLAY_SH1107_128_I2C  // GME128128-02, 128x128 I2C (варіант SH1107)
// #define DISPLAY_SSD1327_128_I2C // GME128128-02, 128x128 I2C (варіант SSD1327)
// #define DISPLAY_ST7567_SPI      // Open-Smart 1.8" 128x64 SPI
// #define DISPLAY_PCD8544_SPI     // Nokia 5110 (PCD8544) 84x48 SPI
```

Роздільність та інтерфейс підставляються автоматично.

**Піни I2C** (КРОК 2a): `SDA=21`, `SCL=22`, адреса `0x3C`.
Для швидшого рендера на GPIO21/22 розкоментуйте `#define DISPLAY_HW_I2C`.

**Піни SPI** (КРОК 2b, для ST7567 / Nokia 5110): апаратний SPI ESP32 —
`SCK=18`, `MOSI=23` (до CLK/DIN дисплея), керуючі `CS=5`, `DC=17`, `RST=16`.

> GME128128-02 буває на контролері SH1107 **або** SSD1327 — якщо картинка
> спотворена, перемкніть між цими двома опціями.
> На Nokia 5110 (84×48) частина нижніх рядків не влазить фізично — усі
> дані краще дивитись у веб-інтерфейсі.

## 4. Прошивка

1. Оберіть плату ESP32 і порт.
2. Скомпілюйте та завантажте скетч (`motorola-battery-reader-web.ino`).
3. **Завантажте файли у SPIFFS** — вміст теки `data/` (`index.html`,
   `logo.png`): меню *Tools → ESP32 Sketch Data Upload* (плагін
   `arduino-esp32fs`/`arduino-littlefs`; для SPIFFS — ESP32FS).
   Без цього кроку веб-сторінка не відкриється.

## 5. Використання

1. Підключіться до Wi-Fi **`BatteryReader_moto`**, пароль **`12345678`**.
2. **Captive-portal**: телефон/ПК сам запропонує «Вхід у мережу» і відкриє
   сторінку. Якщо ні — відкрийте `http://192.168.4.1`.
3. Вкладки: **Огляд** (заряд %, напруга, температура, модель, знос,
   справжність), **Дані** (модель/серійний, DS2438, повні hex-дампи),
   **Прошивка** (зчитати/зберегти, записати/відновити, скидання).
4. Пароль для запису/скидання — `admin123` (`ADMIN_PASSWORD` у `settings.h`).

### Кнопки на пристрої
- **«Вперед» коротко** — наступна сторінка меню.
- **«Вперед» довго (0.8 с)** — повторне зчитування АКБ.
- **«Назад» коротко** — попередня сторінка; на сторінці «Скидання» —
  взвід, друге натискання — підтвердження скидання.

## 6. Скидання лічильників (рекалібрування)

Обнуляє лічильники заряду/розряду (CCA/DCA), таймер і знос (ємність→100%)
для подальшого калібрування на оригінальній зарядці IMPRES. Калібрувальний
offset-регістр DS2438 зберігається. Доступно з веба (вкладка «Прошивка»,
під паролем) та з меню дисплея (сторінка «Скидання», подвійне «Назад»).

## 7. Одиниці вимірювання

Лічильники заряду показані в **мА·год** (не «сирими» числами):
`мА·год = регістр × 0.4882 / Rsense`. Резистор струму задається в
`settings.h` (`DS2438_RSENSE_OHM`, за замовч. 0.025 Ом) — підберіть так,
щоб ємність сходилась з паспортною (типово 20…40 мОм).

## ⚠️ Важливо (не «виправляти»!)

У `battery_reader.h` коди сімейства 1-Wire для ЦЬОГО заліза —
`DS2433_ID = 0xA3`, `DS2438_ID = 0xA6` (перевірено осцилографом). Це
**не** стандартні 0x23/0x26 з даташита. Заміна на «стандартні» ламає
зчитування — не чіпайте.
